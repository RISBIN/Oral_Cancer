## User Authentication Sequence Diagram

```mermaid
sequenceDiagram
    participant U as üë§ User
    participant B as üåê Browser
    participant D as Django
    participant DB as üíæ Database
    participant S as üîê Session

    U->>B: Enter credentials
    B->>D: POST /accounts/login/
    D->>D: Validate CSRF token
    D->>DB: Query user by email
    DB-->>D: User record
    D->>D: Verify password hash

    alt ‚úÖ Valid Credentials
        D->>S: Create session
        S-->>D: Session ID
        D->>DB: Update last_login
        D-->>B: Set session cookie
        B-->>U: ‚úÖ Redirect to dashboard
    else ‚ùå Invalid Credentials
        D-->>B: Return error
        B-->>U: ‚ùå Show error message
    end
```
